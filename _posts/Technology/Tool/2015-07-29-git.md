## 简介（未完成）

git reset --hard commit_id(可用 git log –oneline 查看) ###本地代码回退

git push origin HEAD --force ###远程提交回退

或者

git reset --hard HEAD~1

git push --force

git revert是用一次新的commit来回滚之前的commit，git reset是直接删除指定的commit。
